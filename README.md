# shot-predictor-serie-a-2026
Analisi e previsione tiri Serie A in Python. Database integrato per massima portabilità su Colab senza file CSV. Usa la Distribuzione di Poisson, indici di difficoltà difensiva e normalizzazione nomi per calcolare le probabilità (1+, 2+, 3+) con report testuali e grafici. Uno strumento di Data Science sportiva pronto all'uso.
